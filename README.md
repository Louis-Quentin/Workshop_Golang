# 1: Créer un projet en Go
  - Télécharger Go: 
    - https://go.dev/dl/go1.20.linux-amd64.tar.gz
    
  - Installer Go:
    - rm -rf /usr/local/go && tar -C /usr/local -xzf go1.20.linux-amd64.tar.gz
    - export PATH=$PATH:/usr/local/go/bin
  
  - Vérifier l'installation:
    - go version
